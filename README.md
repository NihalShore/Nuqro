NUQRO

A Python-based CLI assessment platform that measures IQ, EQ, SQ, and AQ with age-normalised scoring.

NUQRO runs entirely in your terminal – no login, no graphics, no internet after download. It asks for your age once, then uses that to normalise scores so you can fairly compare yourself to people of the same age.

Features

- Four tests – Intelligence (IQ), Emotional Intelligence (EQ),Social Intelligence (SQ), Adversity Quotient (AQ)
- Age-normalised scoring – Z-score method with norm tables
- Sub-score breakdowns – See your EQ dimensions, SQ factors, and AQ CORE scores
- Resume any test – Stop halfway and pick up where you left off
- Persistent history – All results saved in data/results.json
- CSV export – One-click export of your entire history
- Optional timer – Encourages spontaneous answers for more accurate results
- Lie scale (social desirability) – Detects response bias
- ASCII charts and coaching tips – Visual feedback and personalised advice
- No external dependencies – Uses only Python standard library

Quick start

# Clone or download the repository
cd nuqro

# (Optional) create a virtual environment
python -m venv venv
source venv/bin/activate   # On Windows: .\venv\Scripts\activate

# Generate the norm tables (run once)
python generate_norms.py

# Launch the program
python main.py

Usage

After starting, you will be asked for your name and age. Then choose which tests to take (or run all four). Answer each question with A, B, C, or D. If you enable the timer, you will have a limited number of seconds per question – if time runs out, the question is skipped and scores a neutral value.

After the main tests, you may optionally take a short lie scale (truthfulness check) that helps interpret your EQ/SQ/AQ results.

At the end, NUQRO shows:
- Your adjusted score (age-normalised)
- Raw score (number correct for IQ, percentage for others)
- Sub-score breakdown (for EQ, SQ, AQ)
- A coaching tip for your lowest-scoring dimension
- A truthfulness warning if the lie scale indicates bias

You can then save the report as a text file or JSON, export your full history to CSV, or view past results.

Command-line options

# Run only one test
python main.py --test IQ

# Show explanations for IQ answers
python main.py --explain

# Enable timer with custom seconds
python main.py --timer 20

# View past results without running tests
python main.py --history

# Export all history to CSV
python main.py --export

How scoring works

- IQ – Number of correct answers, converted to 0-100 raw, then age-normalised to a standard IQ scale (mean=100, SD=15).
- EQ / SQ / AQ – Likert scale answers (1-5) summed and converted to 0-100 raw. Sub-scores computed per dimension/factor. Final adjusted score is age-normalised and clamped to 0-100.

Norm tables (means and standard deviations for each age group) are stored in data/norms.json. By default they are generated by generate_norms.py from a simulated virtual population – see docs/ for more details.

Limitations

NUQRO is a demonstration project for Stanford Code in Place. It is not a clinically validated instrument. The norm tables are simulated, the question banks are small (10 per test), and there is no full control for response bias (though a lie scale is included). Do not use it for real psychological assessment.
